╔════════════════════════════════════════════════════════════════╗
║        TUNNEL GATEWAY - STANDALONE (NO CONTAINER)             ║
║                    60 Second Quickstart                       ║
╚════════════════════════════════════════════════════════════════╝

1️⃣  COPY THE FILES
   ───────────────────────────────────────────────────────────
   Copy these 2 files anywhere on your Ubuntu machine:
     • tunnel-gateway-local.sh   (the script)
     • local-tunnels.conf        (your config)


2️⃣  EDIT CONFIG
   ───────────────────────────────────────────────────────────
   nano local-tunnels.conf

   Example:
   ────────────────────────────────────────────────────────────
   [mysql-prod]
   sshAddress = db.production.com
   sshUsername = youruser
   sshPassword = yourpass
   localPort = 3306
   remoteHost = localhost
   remotePort = 3306

   [kafka-prod]
   sshAddress = kafka.production.com
   sshUsername = youruser
   sshKeyFile = ~/.ssh/kafka_key
   localPort = 9092
   remoteHost = localhost
   remotePort = 9092
   ────────────────────────────────────────────────────────────


3️⃣  RUN IT
   ───────────────────────────────────────────────────────────
   ./tunnel-gateway-local.sh local-tunnels.conf

   Output:
   ════════════════════════════════════════════════════════════
   ✓ All dependencies installed
   ✓ Started (PID: 123456)
   ✓ 2 tunnel(s) active

   Tunnels are running!
   Press Ctrl+C to stop all tunnels
   ════════════════════════════════════════════════════════════


4️⃣  USE IN YOUR CODE
   ───────────────────────────────────────────────────────────
   MySQL:      jdbc:mysql://127.0.0.1:3306/mydb
   Kafka:      bootstrap.servers=127.0.0.1:9092
   PostgreSQL: jdbc:postgresql://127.0.0.1:5432/mydb
   Redis:      127.0.0.1:6379


5️⃣  STOP TUNNELS
   ───────────────────────────────────────────────────────────
   Press Ctrl+C in the terminal where script is running
   All tunnels stop automatically


═══════════════════════════════════════════════════════════════

📌 TIPS

• Keep script running in a separate terminal while you work
• Or use tmux/screen to run it in background
• Config uses same INI format as container version
• All tunnels listen on 127.0.0.1 (localhost only)
